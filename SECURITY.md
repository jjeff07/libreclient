# Security Policy

## Supported Versions

| Version  | Supported |
|----------|-----------|
| latest   | ✅         |
| < latest | ❌         |

Only the latest released version receives security updates.

## Reporting a Vulnerability

**Please do not open a public issue for security vulnerabilities.**

Instead, report vulnerabilities privately
via [GitHub Security Advisories](https://github.com/jjeff07/libreclient/security/advisories/new).

You can expect:

- Acknowledgement within **7 days**
- A fix or mitigation plan within **14 days** for confirmed issues
- Credit in the release notes (unless you prefer anonymity)

## Scope

This policy covers the `libreclient` Python package. Issues related to the LibreNMS API server itself should be reported
to the [LibreNMS project](https://github.com/librenms/librenms/security).
